# symbols
extract symbols from file 

usage:

	symbols [--all] [--stdin] [--threads]

	flags:

	--all
		count occurrences of ALL characters,
		not just symbols
	
	--stdin
		read file list from stdin
	
	--threads
		run program in multithreaded manner
